Description

Here you will find the description of the JavaScript revision 01 exercise. The goal is simple. Create a form that does one simple thing: it collects emails.

It looks like this, and this is what it does:

subscribe

Get connected to your future

As you have noticed, you have a working example here. So you have the option: either you try to copy it as accuratly as possible without looking at the code or you copy the code. Either way, if you get it to work exactly as the example above, you will learn something.

How do I even... ?

I'm no designer, so I look online for examples I like and try to mimick them as best as possible. Here's the example I used: https://www.codehim.com/bootstrap/bootstrap-email-subscription-form/

I chose this example explicitly because it also has the design in code. I didn't look at it, because I don't want to let it influence my thinking. Only afterwards did I look at the code to compare what I liked best about their code and what I liked more about my own.

So not only do you learn to code by mimicking, also do you learn by comparing. By doing so experience will come automatically.

With experience comes craftmanship and routine. You might think this revision exercise is a daunting task. Don't be alarmed, it is. It's also a learning experience. Set your own goals and see how far you can reach them. Even if you never, ever plan on becoming a developer, you get the opportunity to experience how it is to be one, even just momentarily.

You will meet many developers in your life, in many shapes and forms. You will think of this course when you do and you'll be able to make a connection... if you do this exercise right at least.
